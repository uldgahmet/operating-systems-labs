#Example04-Title

##Purpose
It is the process of reading from a file twice in succession and printing the read data to the screen.

##System Calls/ Function Used
open():Open the file, place the file cursor at the beginning.
read():Reading data from a file.
write():Writing data to stdout (screen).

##How it Works
It is based on reading 10 bytes from a file twice in sequence and writing the read data to the screen.

##How to Compile and Run
&gcc main.c -o app
&./app
