##Purpose
Its main purpose is to demonstrate how the write() system call works and to show that the number of bytes written is the return value.

##System Calls/ Function Used
The write() function is a system call included in the POSIX standard. It writes raw data to a file descriptor.
printf() is a standard C library function found in stdio.h. Its function is to produce formatted output.
