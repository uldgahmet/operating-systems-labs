#Example03-Title

##Purpose
The purpose of this program is to read data from standard input and append it to a file.

In other words, the program receives data from the user and appends it to the end of the file named "target."

##System Calls/ Functions Used
read():Reading data from file, terminal or stdin.
open():Opening a file and obtaining a file descriptor.
write():Writing data to a file or output device.

##How it Works
It is based on reading data from the user (stdin) and appending the read data to the end of a file.

##How to Compile and Run
&ggc main.c -o app
&./app
